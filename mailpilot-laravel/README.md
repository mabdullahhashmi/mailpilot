# MailPilot - Laravel Edition

## About MailPilot

MailPilot is an email marketing and warmup tool built with Laravel. This is the modern Laravel version of the original PHP tool.

## Installation

1. Clone the repository
2. Run `composer install`
3. Copy `.env.example` to `.env` and configure your database
4. Run `php artisan key:generate`
5. Run migrations: `php artisan migrate`
6. Start the application: `php artisan serve`

## Requirements

- PHP 8.1+
- MySQL 5.7+
- Composer
